# 🛒 Django Online Shop

A full-featured e-commerce platform built with Django.

## ✨ Features (so far)

- User authentication (register, login, logout) with OTP via SMS
- Product listing with category & price filters
- Product detail page with size-based pricing
- Shopping cart & checkout with Zarinpal payment
- Custom admin panel for managing products and orders
- Background tasks with Celery + Celery Beat (managed by Supervisor)
- External file storage using ArvanCloud bucket
- Database: SQLite (will migrate to PostgreSQL later)
